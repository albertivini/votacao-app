
# Aplicativo WEB de votação 

Recursos utilizados:

- Node.Js
- Express
- MongoDB
- Passport
- Joi
- Nodemailer
- EJS

## Projeto

Projeto de aplicativo WEB inspirado no paredão do Big Brother Brasil.

- Criar usuário
- Validar usuário;
- Votar em determinado participante;
- Visualizar a contagem de votos;
